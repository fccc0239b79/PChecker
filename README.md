# PChecker
ComPChecker is a desktop application for building PCs, the application allows a user to build an computer and see if parts are compatible with each other.
This project was developed for INSE(Introduction to Software Engineering) at University of Portsmouth.

This application was developed by Grzegorz Lindert and Pawel Szymczyk.
